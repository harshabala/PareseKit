## ParseKit v0.2.10

UI craft and motion polish — the menu-bar app should feel faster and clearer.

### What's new

- **Smoother motion** — ease-out enters/exits, no panel blur jank, shorter reduced-motion fades
- **Escape is instant** — keyboard dismiss of Settings / History skips fly animations
- **Convert blocked reason** — when Convert is disabled, a short caption explains why (e.g. pick an output folder)
- **Press & hover craft** — segment controls and back buttons respond with light press scale (fine pointers only)
- **Faster dependency check** — File Support no longer waits an extra beat before recheck
- **Progress polish** — title / error lines crossfade instead of hard-swapping
- **Focus trap** — Tab stays inside Settings, History, and About; focus restores on close
- **Empty history** — clear path back to convert
- **Settings open** — light scale from the gear origin

### Install

1. Download **`ParseKit_0.2.10_aarch64.dmg`** (Apple Silicon, macOS 12+)
2. Open → drag **ParseKit** to **Applications**
3. Open from Applications → icon in the **menu bar**

Gatekeeper one-liner (first launch):

```bash
xattr -cr /Applications/ParseKit.app
```

[Install guide](https://github.com/harshabala/parsekit/blob/master/docs/INSTALL.md)

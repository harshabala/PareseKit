## ParseKit v0.2.9

Menu-bar converter: PDFs, Office, and images → clean Markdown **on your Mac**.

### Highlights

- Reliable signed DMG + in-app updater pipeline
- Menu-bar panel polish and motion refinements
- Settings / About layout fixes

### Install

1. Download **`ParseKit_0.2.9_aarch64.dmg`** (Apple Silicon, macOS 12+)
2. Open → drag **ParseKit** to **Applications**
3. Open from Applications → look for the icon in the **menu bar**

First launch blocked?

```bash
xattr -cr /Applications/ParseKit.app
```

Full steps: [Install guide](https://github.com/harshabala/parsekit/blob/master/docs/INSTALL.md)
